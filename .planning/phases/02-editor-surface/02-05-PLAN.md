1. Create `src/lib/pages/page-crud.ts` — pure functions, no React hooks:

   ```typescript
   import type { Page, Project } from '@/types/project';
   import { FORMATS } from '@/constants/formats';
   ```

   **`addPage(project: Project): { updatedProject: Project; newPageIndex: number }`**
   - Create new page: `{ id: crypto.randomUUID(), elements: [], background: '#FFFFFF' }`
   - Append to project.pages
   - Return updated project and the index of the new page

   **`duplicatePage(project: Project, sourceIndex: number, projectId: string): { updatedProject: Project; newPageIndex: number }`**
   - Create new page with new id and same background
   - Copy sessionStorage content: read `dessy-generated-page-${projectId}-${sourceIndex}`, write to `dessy-generated-page-${projectId}-${newIndex}` where newIndex = project.pages.length
   - Append new page to project.pages
   - Return updated project and new page index

   **`deletePage(project: Project, pageIndex: number, projectId: string): { updatedProject: Project; newCurrentIndex: number }`**
   - Guard: if `project.pages.length <= 1`, throw Error("Cannot delete the last page")
   - Remove page from array
   - Remove sessionStorage key for that page
   - Remap all sessionStorage keys after the deleted index (shift down by 1):
     ```
     for (let i = pageIndex; i < project.pages.length - 1; i++) {
       const next = sessionStorage.getItem(`dessy-generated-page-${projectId}-${i + 1}`);
       if (next) sessionStorage.setItem(`dessy-generated-page-${projectId}-${i}`, next);
     }
     sessionStorage.removeItem(`dessy-generated-page-${projectId}-${project.pages.length - 1}`);
     ```
   - New current index: min(pageIndex, updatedPages.length - 1)
   - Return updated project and new current index

   **`reorderPages(project: Project, fromIndex: number, toIndex: number, projectId: string): Project`**
   - Read ALL page JSONs into temp array:
     ```
     const pageJsons: (string | null)[] = project.pages.map((_, i) =>
       sessionStorage.getItem(`dessy-generated-page-${projectId}-${i}`)
     );
     ```
   - Reorder the pages array (splice from, insert at to)
   - Reorder the pageJsons array the same way
   - Re-write all sessionStorage keys:
     ```
     pageJsons.forEach((json, i) => {
       const key = `dessy-generated-page-${projectId}-${i}`;
       if (json) sessionStorage.setItem(key, json);
       else sessionStorage.removeItem(key);
     });
     ```
   - Update currentPageIndex if it was the moved page
   - Return updated project

   **`ensureFormatPageCount(project: Project): Project`**
   - Look up `FORMATS[project.meta.format]`
   - If format not found (custom), return project unchanged
   - Required pages = format.pages
   - If `project.pages.length < required`: add blank pages until count matches
   - If `project.pages.length >= required`: do nothing (user may have added extras)
   - Return updated project
