1. Create `src/hooks/useSelectedObject.ts`:
   - Export `ObjectSnapshot` interface:
     ```typescript
     export interface ObjectSnapshot {
       id: string;
       type: 'text' | 'image' | 'shape' | 'colorBlock' | null;
       x: number; y: number; w: number; h: number; // all in mm via pxToMm
       rotation: number; // degrees
       opacity: number; // 0-100 (display scale)
       fill: string | object | null; // string for solid, object for gradient
       stroke: string | null;
       strokeWidth: number;
       strokeDashArray: number[] | null;
       shadow: { offsetX: number; offsetY: number; blur: number; color: string } | null;
       rx: number; // corner radius in px
       ry: number;
       // text-only properties
       fontFamily?: string;
       fontSize?: number;
       fontWeight?: number | string;
       lineHeight?: number;
       charSpacing?: number;
       textAlign?: string;
       // image-only
       fitMode?: string;
       // swatch linking
       swatchId?: string | null;
       presetId?: string | null;
     }
     ```
   - Export `useSelectedObject()` hook that:
     - Gets `canvasRef` from canvasStore via selector
     - Listens to canvas events: `selection:created`, `selection:updated`, `object:modified`, `selection:cleared`
     - On each event, reads `canvas.getActiveObject()`, extracts properties into ObjectSnapshot using `pxToMm()` for position/size
     - Returns `ObjectSnapshot | null`
     - Handles shadow by checking `obj.shadow` is a Shadow instance and reading `.offsetX`, `.offsetY`, `.blur`, `.color`
     - For multi-selection (`canvas.getActiveObjects().length > 1`), return a snapshot with `type: null` and numeric fields as NaN to signal "mixed"

2. Create `src/hooks/useCanvasLayers.ts`:
   - Export `LayerItem` interface:
     ```typescript
     export interface LayerItem {
       id: string;
       name: string;
       type: string; // customType value
       visible: boolean;
       locked: boolean;
     }
     ```
   - Export `useCanvasLayers()` hook that:
     - Gets canvasRef from canvasStore
     - Maintains `layers: LayerItem[]` state — reversed z-order (top of list = front of canvas)
     - Refreshes on: `object:added`, `object:removed`, `object:modified`, `after:render`
     - Export `moveLayer(fromIndex: number, toIndex: number)` — converts reversed indices back to canvas z-index, calls `canvasRef.moveTo(obj, canvasIndex)` + `renderAll()`
     - Export `toggleVisibility(id: string)` — find object by id, toggle `visible`, call `renderAll()`
     - Export `toggleLock(id: string)` — find object by id, toggle `selectable` and `evented` (both false when locked), set custom `locked` property, call `renderAll()`
     - Export `renameLayer(id: string, name: string)` — find object by id, set custom `name` property
     - Export `selectLayer(id: string)` — find object by id, call `canvas.setActiveObject(obj)` + `renderAll()`, also call `useCanvasStore.getState().setActiveTool('select')`
