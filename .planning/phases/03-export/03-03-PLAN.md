Create `src/lib/export/coreldraw-export.ts`:

**SVG Export:**
1. `exportCorelDrawSVG(canvas, projectId, pages, currentPageIndex, docWidthMm, docHeightMm, projectName)`:
   - Uses collectAllPageData
   - For each page: create temp Canvas, loadFromJSON, call `canvas.toSVG()` with mm width/height and viewBox
   - Single page: download as .svg
   - Multi-page: ZIP with one .svg per page
   - Dispose temp canvas after each

**VBA Macro Export:**
2. `generateCorelDrawMacro(pages, docWidthMm, docHeightMm)` — builds the VBA string:
   - Header comment
   - `Sub CreateLayout()`
   - `Set doc = CreateDocument()`, `doc.Unit = cdrMillimeter`
   - For each page: `doc.AddPages 1` (after first), `page.SetSize`, `page.Activate`
   - Process objects with Y-axis inversion: `corelY = docHeightMm - (topMm + heightMm)`

3. `generateCorelObject(obj, pageIdx, docHeightMm)` — maps Fabric.js objects:
   - **text** → `ActiveLayer.CreateParagraphText x1, y1, x2, y2, "text"` (paragraph text for wrapping) + font/size settings
   - **shape (rect) / colorBlock** → `ActiveLayer.CreateRectangle2 x, y, w, h` + fill color via `shape.Fill.UniformColor.RGBAssign r, g, b`
   - **shape (circle)** → `ActiveLayer.CreateEllipse x1, y1, x2, y2` + fill color
   - **shape (line)** → `ActiveLayer.CreateLineSegment x1, y1, x2, y2` + outline color
   - **image** → `ActiveLayer.CreateRectangle2` with comment for image placement

4. `exportCorelDrawVBA(canvas, projectId, pages, currentPageIndex, docWidthMm, docHeightMm, projectName)`:
   - Uses collectAllPageData
   - Calls generateCorelDrawMacro
   - Downloads as .bas file
