1. Install dependencies:
   ```bash
   npm install file-saver jszip
   npm install -D @types/file-saver
   ```

2. Create `src/lib/export/export-utils.ts`:
   - `collectAllPageData(canvas, projectId, pages, currentPageIndex)` — returns array of `{ pageIndex, canvasJSON, pageId, background }` by reading current canvas + sessionStorage for other pages
   - `hexToRgb(hex: string): [number, number, number]` — converts hex color to RGB array [0-255]
   - `pxToMm(px: number): number` — converts 72-DPI pixels to mm (px * 25.4 / 72)
   - `mmToPx(mm: number): number` — inverse
   - Export a `DPI_MULTIPLIERS` constant: `{ 72: 1, 150: 2.08, 300: 4.17 }`

3. Create `src/lib/export/raster-export.ts`:
   - `exportRasterPages(canvas, projectId, pages, currentPageIndex, docWidth, docHeight, format, dpi, projectName)`:
     - Uses `collectAllPageData` to gather all pages
     - For each page: create temp Fabric Canvas, loadFromJSON, strip before:render listeners (AligningGuidelines workaround from PagesPanel), call `toCanvasElement(multiplier)`, then `HTMLCanvasElement.toBlob()`
     - If single page: download directly with file-saver `saveAs(blob, name)`
     - If multi-page: bundle into ZIP with jszip, then `saveAs(zipBlob, name.zip)`
     - Dispose temp canvas after each page
  </action>
