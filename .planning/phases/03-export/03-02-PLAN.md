Create `src/lib/export/indesign-export.ts`:

1. `generateInDesignScript(pages, meta, docWidthMm, docHeightMm)` — builds the full .jsx string:
   - Header comment with project name and date
   - `app.scriptPreferences.measurementUnit = MeasurementUnits.MILLIMETERS;`
   - Document creation with page dimensions
   - `addColor(name, r, g, b)` helper function (try existing, catch and create)
   - For each page: add page if not first, then iterate objects
   - Alert on completion with font substitution note

2. `generateObjectScript(obj, pageIdx)` — maps each Fabric.js object to ExtendScript:
   - **text** → `page.textFrames.add()` with geometricBounds, contents, font properties (with try/catch fallback), color, justification
   - **image** → `page.rectangles.add({contentType: ContentType.GRAPHIC_TYPE})` with TODO comment for image placement
   - **shape (rect)** → `page.rectangles.add()` with fill color
   - **shape (circle)** → `page.ovals.add()` with fill color
   - **shape (line)** → `page.graphicLines.add()` with path points, stroke color/weight
   - **shape (triangle)** → `page.polygons.add()` with 3 path points
   - **colorBlock** → `page.rectangles.add()` with fill color
   - **group** → flatten children, process individually, add group comment
   - Handle rotation: `obj.rotationAngle = -fabricAngle`
   - Handle stroke on shapes: strokeColor + strokeWeight

3. `mapTextAlign(align)` — maps left/center/right/justify to InDesign Justification enum

4. `exportInDesign(canvas, projectId, pages, currentPageIndex, docWidthMm, docHeightMm, projectName)`:
   - Uses collectAllPageData
   - Calls generateInDesignScript
   - Downloads as .jsx file via file-saver saveAs
