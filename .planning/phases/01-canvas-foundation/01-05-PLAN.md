    1. Create `src/hooks/useHistory.ts`:
       - Follow Pattern 5 from research exactly, with these additions:
       - Two ref arrays: `undoStack` and `redoStack`, each holding `string` (serialized JSON)
       - `currentState` ref holding the current canvas state as string
       - `isProcessing` ref boolean to prevent recursive captures during restore
       - `maxSteps = 50` constant (CANV-07)

       `captureState(canvas: Canvas)`:
       - If `isProcessing.current` is true, return immediately (guard against recursive capture)
       - Serialize: `JSON.stringify(canvas.toDatalessJSON(CUSTOM_PROPS))` — import CUSTOM_PROPS from element-factory
       - Push `currentState.current` onto `undoStack` array
       - If `undoStack.length > maxSteps`, `undoStack.shift()` (drop oldest)
       - Clear `redoStack` (new action invalidates redo)
       - Set `currentState.current` to new snapshot
       - Update Zustand: `useCanvasStore.getState().setUndoRedo(undoStack.length > 0, false)`

       `undo(canvas: Canvas)`:
       - If `undoStack` empty: show toast "Nothing left to undo", return
       - Set `isProcessing.current = true`
       - Push `currentState.current` onto `redoStack`
       - Pop from `undoStack` into `currentState.current`
       - `await canvas.loadFromJSON(JSON.parse(currentState.current))` — MUST await (Fabric.js 7 async)
       - `canvas.renderAll()`
       - Set `isProcessing.current = false`
       - Update Zustand: `setUndoRedo(undoStack.length > 0, redoStack.length > 0)`

       `redo(canvas: Canvas)`:
       - Mirror of undo but popping from redoStack, pushing to undoStack

       `bindHistory(canvas: Canvas)`:
       - Initialize `currentState.current` with initial canvas state
       - `canvas.on('object:added', () => captureState(canvas))`
       - `canvas.on('object:removed', () => captureState(canvas))`
       - `canvas.on('object:modified', () => captureState(canvas))`
       - NOTE: `object:modified` fires on mouseup after drag (one step per drag — per CONTEXT.md decision)

       Return: `{ undo, redo, captureState, bindHistory, canUndo, canRedo }`

    2. Integrate in `src/hooks/useFabricCanvas.ts`:
       - After canvas creation, call `bindHistory(canvas)`
       - Store `undo` and `redo` functions in a ref or return them for keyboard shortcut access
       - Expose history functions via a module-level ref or by passing them to the canvas store

    3. Wire Header undo/redo buttons in `src/components/editor/ui/Header.tsx`:

    1. Create `src/hooks/useKeyboardShortcuts.ts`:
       - Accept `canvas: Canvas | null` and history functions `{ undo, redo }`
       - Register global `keydown` event listener via `useEffect` on `document`
       - Clipboard ref: `clipboardRef = useRef<FabricObject[]>([])` for copy/paste buffer

       **Tool shortcuts (no modifiers, single key):**
       - `v` → setActiveTool('select')
       - `t` → setActiveTool('text')
       - `r` → setActiveTool('rect')
       - `c` (no modifier) → setActiveTool('circle') — BUT only when no text is being edited
       - `l` → setActiveTool('line')
       - `i` → setActiveTool('image')
       - `h` → setActiveTool('hand')
       - IMPORTANT: Skip all single-key shortcuts if `canvas.getActiveObject()?.isEditing` is true (user is typing in a Textbox)

       **Edit shortcuts (with modifiers):**
       - `Ctrl+Z` → call `undo(canvas)` — prevent default browser undo
       - `Ctrl+Shift+Z` → call `redo(canvas)` — prevent default
       - `Ctrl+C` → copy: `canvas.getActiveObject()?.clone().then(cloned => clipboardRef.current = [cloned])`
         In Fabric.js 7, `.clone()` returns a Promise.
       - `Ctrl+V` → paste: take `clipboardRef.current[0]`, clone it again (so we can paste multiple times), offset by `{left: +10, top: +10}`, add to canvas, set as active
       - `Delete` or `Backspace` → `canvas.remove(canvas.getActiveObject())` if not editing text
       - Arrow keys (no modifier) → nudge 1px: `obj.set({ left: obj.left + dx })`, `canvas.renderAll()`
       - Arrow keys + Shift → nudge 10px
       - `Ctrl+G` → group: `const objs = canvas.getActiveObjects(); const group = new Group(objs); canvas.remove(...objs); canvas.add(group)`
       - `Ctrl+Shift+G` → ungroup: `group.toActiveSelection()`

       **File shortcuts:**
       - `Ctrl+S` → prevent default, trigger save (placeholder — wired in Plan 06)
       - `?` → `useEditorStore.getState().setShortcutsModalOpen(true)`

       **Cleanup:** Remove event listener on unmount
