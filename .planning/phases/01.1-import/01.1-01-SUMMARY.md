## Decisions Made

- Native fetch used for both providers — no @anthropic-ai/sdk or @google/genai needed, keeping bundle lean
- System prompt includes a full concrete JSON example (not just a schema description) to ground the AI output
- validate-then-repair is unconditional: every canvas goes through validation, invalid ones get repaired automatically
- Retry uses a fresh prompt asking for JSON-only rather than a full re-generation, saving tokens

## User Setup Required

Environment variables required before the endpoint works:

```
ANTHROPIC_API_KEY=sk-ant-...   # Claude API key
# OR
GEMINI_API_KEY=...              # Gemini API key

AI_PROVIDER=anthropic           # 'anthropic' (default) or 'gemini'
ANTHROPIC_MODEL=claude-sonnet-4-20250514   # optional override
GEMINI_MODEL=gemini-2.5-flash              # optional override
```

Add to `.env.local`. No dashboard configuration required beyond obtaining API keys.
