**Template JSON structure:** Each template file is a valid Fabric.js canvas JSON that matches the `canvasJSON` field format from `saveProject()`. Structure:

```json
{
  "version": "7.0.0",
  "objects": [
    {
      "type": "Rect",
      "left": 0, "top": 0,
      "width": 595, "height": 842,
      "fill": "#FFFFFF",
      "selectable": false,
      "evented": false,
      "_isDocBackground": true,
      "customType": "background"
    },
    ... more Fabric.js objects (Rect, Textbox, etc.)
  ]
}
```

**CRITICAL: No base64 image data in templates.** Use only Fabric.js primitive objects: `Rect` (colored blocks, shapes), `Textbox` (text frames with placeholder text), `Circle`, `Triangle`. Use `customType` values matching existing element factory: `'colorBlock'`, `'textFrame'`, `'shape'`. Do NOT use `'imagePlaceholder'` with embedded images.

**Create 10 visually distinct placeholder templates** using basic Fabric.js objects — colored rectangles for layout sections, Textbox elements for headings/body, shapes for visual interest. Each template should look like a realistic leaflet skeleton with appropriate colors and typography for its category.

Canvas coordinate system: A4 = 595x842 (at 72dpi), A5 = 420x595, DL = 281x595, Bifold = 595x842 per page (2 pages), Trifold = 595x842 per page (3 pages). Use `mmToPx` at 72dpi: 1mm = 2.834645669px.
